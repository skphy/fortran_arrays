#. assign a array to a variable, say a
  
  .. code-block:: fortran
  	  
	  Real(dp) :: a(10)
	  ! this defines a 1D array of length 10
	  ! dp means double precision i.e. kind type of array;  used to defined the precision and range of the array `a`
	  !
